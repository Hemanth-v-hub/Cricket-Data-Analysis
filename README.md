# Cricket-Data-Analysis
##  Objective
Analyze cricket data to identify player and team performance trends.

## Tools Used
- Excel
- Tableau / Power BI

##  Key Analysis
- Player performance (strike rate, averages)
- Team comparison
- Match-winning factors

##  Key Insights
- Identified top-performing players based on consistency
- Found patterns in winning matches
- Built dashboards for easy visualization

## Outcome
This project helps in understanding data-driven decision-making in sports analytics.
